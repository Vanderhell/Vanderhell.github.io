loxdb_pro

Commercial production modules around the loxdb core — for embedded products that need more than storage.

CommercialPublic API docsC API contracts
Overview

The PRO layer for embedded products

loxdb_pro extends the open-source loxdb core with production-grade modules. The public repository (loxdb_pro_docs) intentionally contains API-level documentation only — the implementation and proprietary validation procedures are not published.

Security & integrity

Security and integrity modules plus runtime safety validation.

Observability

Metrics, logging, monitoring and alerting.

Policy & control

Policy gates, quotas, retention and scheduling.

Data lifecycle

Backup and schema migration.

Replication & transport

Replication and transport framing.

Operations

OTA planning and rollback hooks, CLI tooling, optional SD card and NAND/FTL adapters.

Documentation-first by design

The public repo documents the integration-facing API of the commercial module set. If you are evaluating loxdb_pro for a product, the docs show exactly what the contracts look like before any commercial conversation.
